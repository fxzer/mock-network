# MockNetwork 🌐

**简单、强大、无侵入的 Chrome 网络请求拦截与修改工具**

> 还在为后端接口没写好而发愁吗？
> 还在为测试各种奇葩数据而改代码吗？
> MockNetwork 帮你搞定一切！

## 📖 这个项目是做什么的？

通俗地说，MockNetwork 就像是搭建在你浏览器和服务器之间的一个 **"智能收费站"**。

- **正常情况**: 你的网页发请求给服务器，服务器返回数据给网页。
- **MockNetwork 开启后**:
  1. 网页发出的请求，先经过我们的收费站。
  2. 我们看一眼（**匹配规则**）：
     - 没在名单上的：放行，走正常流程。
     - 在名单上的：**拦截下来！**
  3. 拦截后，你可以：
     - **修改请求**: 把发给服务器的参数改了。
     - **伪造响应**: 根本不发给服务器，直接骗网页说"服务器返回了这个数据"。
     - **修改响应**: 等服务器返回了，悄悄改掉里面的数据再给网页看。

全程不需要改你的网页代码，即装即用！

## ✨ 核心功能

- ✅ **零侵入**: 不需要修改你项目的任何代码。
- ✅ **全覆盖**: 支持 `XMLHttpRequest` 和 `fetch` 请求。
- ✅ **够灵活**: 支持普通链接匹配、正则表达式匹配。
- ✅ **易操作**: 提供侧边栏界面，即改即生效。
- ✅ **Mock支持**: 内置 Mock.js 语法，随机生成数据（姓名、地址、图片...）。

## 🚀 快速开始

### 1. 安装依赖

```bash
pnpm install
# 或者
npm install
```

### 2. 构建项目

```bash
npm run build
```

构建完成后，会生成一个 `dist` 目录（或者在 `src/ui/dist` 生成资源）。

### 3. 安装到 Chrome

1. 打开 Chrome 浏览器，输入 `chrome://extensions/`
2. 打开右上角的 **"开发者模式"** 开关。
3. 点击 **"加载已解压的扩展程序"**。
4. 选择本项目根目录。

### 4. 开始使用

1. 打开任意网页（比如百度）。
3. 侧边栏出现！点击 "+" 号添加一个拦截规则。
4. 刷新页面，见证奇迹。

## 📦 打包与分享

如果你想把这个扩展分享给别人，或者生成一个干净的压缩包：

```bash
# 生成打包文件 (mock-network-extension.zip)
node scripts/pack.js

# 如果你想先自动构建 UI 再打包 (推荐)
node scripts/pack.js -b
```

打包产物位于项目根目录：
- `mock-network-extension.zip`：发送给别人即可。
- `mock-network-extension/`：解压后的文件夹，可直接加载。
